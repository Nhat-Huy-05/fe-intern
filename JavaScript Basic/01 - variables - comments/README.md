
# Variable (Giá trị) và Comment

Tóm tắt các kiến thức nền tảng về JavaScript bao gồm khai báo biến, hàm built-in và các loại toán tử.

## 1. Khai báo biến & Comment
- **Khai báo:** - `var`: Khai báo biến (phạm vi global/function).
  - `let`: Khai báo biến hiện đại, có phạm vi khối (block scope).
- **Comment:**
  - `//`: Comment một dòng.
  - `/* */`: Comment nhiều dòng/khối code.

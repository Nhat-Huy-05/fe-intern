# 2. Các hàm Built-in phổ biến
- `alert()`: Hiển thị hộp thoại thông báo.
- `console`: `log()`, `warn()`, `error()` dùng để gỡ lỗi.
- `confirm()`: Xác nhận người dùng (OK/Cancel).
- `prompt()`: Lấy dữ liệu nhập từ người dùng.
- `setTimeout()`: Thực thi code một lần sau một khoảng thời gian.
- `setInterval()`: Thực thi code lặp lại sau mỗi khoảng thời gian.

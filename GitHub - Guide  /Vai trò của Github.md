## 📌 Vai trò của GitHub

GitHub đóng vai trò là hệ thống quản lý và lưu trữ mã nguồn sử dụng Git. Có thể hiểu đơn giản qua 3 chức năng chính:

* **Lưu trữ (Hosting):** Là "cái kho" an toàn để lưu giữ mã nguồn, giúp truy cập mọi lúc mọi nơi.
* **Quản lý phiên bản (Version Control):** Ghi lại lịch sử thay đổi của code, cho phép quay lại các phiên bản cũ khi cần.
* **Cộng tác (Collaboration):** Giúp nhiều người cùng làm việc trên một dự án, kiểm tra code (Review) và chia sẻ kiến thức.

> **Lưu ý:** GitHub là nơi lưu trữ và quản lý code, không phải là nơi trực tiếp thực thi (chạy) code JavaScript trừ khi được cấu hình thêm các dịch vụ bổ trợ.

React JS (hay React) là một thư viện JavaScript được phát triển bởi Meta (Facebook) để xây dựng giao diện người dùng (UI) một cách hiệu quả và linh hoạt.
Những đặc điểm chính:
1. `Thư viện View`:

React tập trung vào phần "View" trong mô hình MVC
Giúp tạo các thành phần UI tái sử dụng được

2. `Component-Based`:

Chia giao diện thành những thành phần (components) nhỏ, độc lập
Mỗi component có thể quản lý trạng thái (state) riêng

3. `Virtual DOM`:

React sử dụng Virtual DOM - một bản sao ảo của DOM thực
Tăng hiệu suất bằng cách chỉ cập nhật những phần thay đổi

4. `Declarative (Khai báo)`:

Mô tả giao diện sẽ như thế nào, React sẽ tự cập nhật
Dễ hiểu và dễ debug hơn Imperative

5. `JSX`:

Cho phép viết HTML-like syntax trong JavaScript
Giúp code dễ đọc và dễ bảo trì hơn

Ví dụ cơ bản:
```
jsxfunction HelloWorld() {
  return <h1>Xin chào React!</h1>;
}
```
**Ưu điểm**:

✅ Hiệu suất cao (Virtual DOM)

✅ Dễ học và dễ sử dụng

✅ Cộng đồng lớn, nhiều tài liệu

✅ Có thể dùng cho web và mobile (React Native)

✅ Tái sử dụng code hiệu quả

React được sử dụng rộng rãi trong các ứng dụng web hiện đại như Facebook, Instagram, Netflix, Airbnb, v.v.
